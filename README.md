# Cara-Cepat-Mengetik-Angka-Besar-di-Excel
Mini project menggunakan Microsoft Excel untuk memahami cara cepat mengetik angka besar dengan notasi ilmiah (e). Teknik ini mempermudah input angka ribuan hingga triliunan tanpa mengetik banyak nol, sehingga lebih efisien dan praktis dalam pengolahan data.
